What is Ainos:

AINOS is basically a chatbot. A software program that can read messages and type replies.
AINOS will take your place on WhatsApp Web to chat with the contacts you select. You can still chat on WhatsApp Mobile.
AINOS gives pre-defined automatic replies to specific phrases.
AINOS is not a hack.
AINOS does not store data or transfer data somewhere else. Your chat data stays with you.
AINOS is not linked to any online database. Database files are installed with AINOS on your PC.
AINOS uses these files to understand messages and give suitable replies.

How is AINOS different from Siri, Alexa, or Google Assistant:

Siri, Alexa or Assistant are highly developed AI systems with superior Natural Language Processing capacities than AINOS. AINOS is a simple AI-based chatbot.
You can chat with Siri, Alexa or Assistant. However, Siri, Alexa or Assistant cannot chat for you.
AINOS can. AINOS will chat for you.

Why should you download AINOS:

You can have your own chatbot to chat for you.
You can teach AINOS specific replies to certain messages and phrases.
AINOS can chat when you are busy or away.

How to activate your own AINOS:

Download and install the latest AINOS.exe file.
Open the AINOS folder.
Run the ainos.exe file.
A Command Prompt Window and WhatsApp Web on Chrome will open.
Scan the QR Code from your WhatsApp App to access WhatsApp Web.
Type in the Command Prompt Window the Name of the Contact or Group (from recent chats).
Press ENTER in the Command Prompt Window.
AINOS will open that chat and wait silently for a message from the other side.
When a message is sent, AINOS will read that and reply accordingly.
To deactivate AINOS, close the Command Prompt Window. This will also close the Chrome Window opened by AINOS.

How to customize your own AINOS:

Open the ainos_csv_database folder.
Open the ainos_alpha.csv file as a Spreadsheet.
Add your own expected phrases (in the first column) and corresponding replies by AINOS (in the second column).
Save and exit the file.
AINOS will respond to the new phrases (if present in the messages from the other side) with your new replies.
